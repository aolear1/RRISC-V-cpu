# RRISC-V-cpu
A reduced reduced intruction set computer that won the 2023 CPEN 211 Lab 7 competition

Handles the following instructions: 
# Control Flow Instructions
- B
- BEQ
- BNE
- BLT
- BLE
- BLX
- BL
- BX
- HALT

# Move Instructions
- LDR
- STR

# ALU Instructions
- ADD
- AND
- CMP

# Move Instructions
- MOV
- MVN

# NOP Instructions
- NOP

# Shift Operators
- LSL
- LSR
- ASR
